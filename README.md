# BaseXXX Pro

![License](https://img.shields.io/github/license/RainPPR/base-code-extended) ![Status](https://img.shields.io/badge/status-active-brightgreen)

**BaseXXX Pro** is a modern, high-performance web-based encoder and decoder utility. It is designed with a focus on aesthetics (Glassmorphism) and versatility, supporting a wide array of Base encoding formats ranging from standard Hex and Base64 to custom Chinese character encoding schemes.

> **Note**: This project is client-side only. No data is sent to any server.

### ðŸ”¢ Extensive Encoding Support

BaseXXX Pro supports real-time conversion for the following formats:

- **Standard Formats**
  - **Base16 (Hex)**: Standard hexadecimal encoding.
  - **Base64**: Web-safe standard encoding (RFC 4648).
  - **Base85**: Adobe Ascii85 implementation.

- **Integer/Alphabet Bases**
  - **Base10**: Decimal representation.
  - **Base26**: A-Z (Case Insensitive).
  - **Base36**: 0-9, A-Z (Case Insensitive).
  - **Base52**: A-Z, a-z (Case Sensitive).
  - **Base62**: 0-9, A-Z, a-z (Case Sensitive).
  - **Base94**: All ASCII printable characters.

- **Special/Experimental**
  - **BaseCN-Common**: Chinese Core Characters (Commonly used \u4E00-\u9FFF + \u3007).
  - **BaseCN-Extended**: Full Chinese CJK range coverage.
